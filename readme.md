# Safer Sale

## Purpose

This app was designed to provide consumers an alternative to mainstream online classified ads sites by including a search for safe locations to meet up such as libraries and police stations.

## Built With

- AWS S3
- JavaScript
- Node.js
- Sequelize
- MySQL2
- Express
- Express Session
- Express Handlebars
